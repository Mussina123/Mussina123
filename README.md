# 👋 What I've Been Working On

> Fullstack engineer focused on backend architecture, infrastructure, and
> performance at scale. Lately I've been shifting from primarily frontend work
> into deeper backend and platform engineering.

---

## 🚀 Recent Focus

### ⚙️ Backend & Infrastructure
- Architecting multi-tier caching services to reduce external API costs and
  improve performance at scale
- Building data pipelines to keep cache layers fresh and decouple refresh
  logic from read paths
- Designing backend services with infrastructure-as-code, establishing
  deployment standards and repeatable workflows across teams
- Building custom cloud infrastructure to consolidate vendors and reduce
  operational costs

### 🤖 AI-Assisted Engineering
- Adopting AI tooling as an accelerator—building automated workflows and
  enforcing code standards while retaining ownership of architecture and
  implementation
- Using AI to assist and augment my work, not replace engineering judgment

### 💻 Frontend & Performance
- Building frontend data layers to improve scalability and reduce API costs
- Leading performance optimization work across Core Web Vitals, engagement,
  and SEO
- Owning delivery of high-traffic web applications from concept to production

### 🛠️ Platform & Developer Experience
- Building observability and alerting platforms to improve incident response
  and reliability
- Establishing testing standards to improve coverage and release confidence
- Leading tooling migrations to streamline CI/CD and speed up developer
  feedback loops
